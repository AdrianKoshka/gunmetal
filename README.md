
# Gunmetal

A fork of gunmetal, a theme for ivy.

## Changes:

- Use Fira Sans mostly for Headings
- Use Noto Serif for everything else


## Include Files

This theme supports the following include files:

* `menu`

  This file should contain a list of links.
